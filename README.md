## 📌 Sobre o Projeto

Este portfólio foi desenvolvido com o objetivo de apresentar meus projetos, habilidades técnicas e evolução na área de tecnologia.

O site foi construído utilizando HTML, CSS e JavaScript, com foco em responsividade, interatividade e uma interface moderna.

Além da parte visual, o projeto também conta com funcionalidades dinâmicas, como animações, efeito de digitação, sistema de envio de e-mails e suporte a múltiplos idiomas (Português, Inglês e Espanhol).

---

## ⚙️ Funcionalidades

- Layout moderno e responsivo  
- Animações ao rolar a página  
- Efeito de digitação dinâmica  
- Sistema de envio de e-mails com EmailJS  
- Alternância de idioma (PT, EN, ES)  
- Seção de projetos com links para GitHub  
- Integração com WhatsApp  
- Efeitos visuais com partículas e interação com o mouse
